用法

python PackerFuzzer.py [选项]
选项
  • -h, --help：显示此帮助信息并退出。
  • -u URL, --url=URL：请输入目标站点。
  • -c COOKIE, --cookie=COOKIE：请输入站点的 Cookie。
  • -d HEAD, --head=HEAD：请输入额外的 HTTP 请求头。
  • -l LANGUAGE, --lang=LANGUAGE：请选择语言。
  • -t TYPE, --type=TYPE：请选择扫描模式。
  • -p PROXY, --proxy=PROXY：请输入您自己的代理地址。
  • -j JS, --js=JS：额外的 JS 文件。
  • -b BASEURL, --base=BASEURL：请输入基础 URL。
  • -r REPORT, --report=REPORT：选择报告的类型。
  • -e EXT, --ext=EXT：启用扩展。
  • -f SSL_FLAG, --flag=SSL_FLAG：SSL 安全标志。
  • -s SILENT, --silent=SILENT：静默模式（自定义报告名称）。
  • --st=SENDTYPE, --sendtype=SENDTYPE：HTTP 请求类型，POST 或 GET。
  • --ct=CONTENTTYPE, --contenttype=CONTENTTYPE：HTTP 请求头中的 Content-Type。
  • --pd=POSTDATA, --postdata=POSTDATA：扫描时的 HTTP 请求 Post 数据。
  • --ah=APIHOST, --apihost=APIHOST：API 主机，例如：https://pocsir.com:777/。
  • --fe=FILENAMEEXTENSION, --fileext=FILENAMEEXTENSION：API 文件名扩展名，例如：.json。
