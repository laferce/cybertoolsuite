SpringBoot-Scan 帮助文档

用法

SpringBoot-Scan.exe [选项]
SpringBoot-Scan 的选项
  • -u URL, --url URL：对单一 URL 进行信息泄露扫描。
  • -uf URLFILE, --urlfile URLFILE：读取目标 txt 文件进行信息泄露扫描。
  • -v VUL, --vul VUL：对单一 URL 进行漏洞利用。
  • -vf VULFILE, --vulfile VULFILE：读取目标 txt 文件进行批量漏洞扫描。
  • -d DUMP, --dump DUMP：扫描并下载 SpringBoot 敏感文件（可提取敏感信息）。
  • -p PROXY, --proxy PROXY：使用 HTTP 代理。
  • -z ZOOMEYE, --zoomeye ZOOMEYE：使用 ZoomEye 导出 Spring 框架资产。
  • -f FOFA, --fofa FOFA：使用 Fofa 导出 Spring 框架资产。
  • -y HUNTER, --hunter HUNTER：使用 Hunter 导出 Spring 框架资产。
  • -t NEWHEADER, --newheader NEWHEADER：从 txt 文件中导入自定义 HTTP 头部。
可选选项
  • -h, --help：显示帮助信息并退出。
