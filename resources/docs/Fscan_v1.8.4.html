Fscan 帮助文档

用法

fscan.exe [参数]
选项
  • -br int：暴力破解线程数（默认值：1）。
  • -c string：执行命令（支持 ssh 或 wmiexec）。
  • -cookie string：设置 poc 的 cookie，例如：-cookie rememberMe=login。
  • -debug int：每次记录错误的时间间隔（默认值：60；单位：秒）。
  • -dns：使用 dnslog poc。
  • -domain string：smb 域名。
  • -full：进行全面 poc 扫描，例如：shiro 100 key。
  • -h string：指定要扫描的主机 IP 地址，例如：192.168.11.11 或 192.168.11.11-255 或 192.168.11.11,192.168.11.12。
  • -hash string：哈希值。
  • -hf string：主机文件，例如：-hf ip.txt。
  • -hn string：不扫描的主机，例如：-hn 192.168.1.1/24。
  • -json：以 json 格式输出。
  • -m string：选择扫描类型，例如：-m ssh（默认值：all）。
  • -no：不保存输出日志。
  • -nobr：不进行暴力破解密码。
  • -nocolor：不使用颜色显示。
  • -nopoc：不扫描 web 漏洞。
  • -noredis：不进行 redis 安全测试。
  • -np：不进行 ping 扫描。
  • -num int：poc 扫描速率（默认值：20）。
  • -o string：输出文件（默认值：result.txt）。
  • -p string：选择端口，例如：22 或 1-65535 或 22,80,3306（默认为常见端口列表）。
  • -pa string：添加基于默认端口的端口，例如：-pa 3389。
  • -path string：fcgi、smb 远程文件路径。
  • -ping：使用 ping 替代 icmp。
  • -pn string：不扫描的端口，例如：-pn 445。
  • -pocname string：使用包含指定名称的 poc，例如：-pocname weblogic。
  • -pocpath string：poc 文件路径。
  • -portf string：端口文件。
  • -proxy string：设置 poc 代理，例如：-proxy http://127.0.0.1:8080。
  • -pwd string：密码。
  • -pwda string：添加基于默认密码的密码，例如：-pwda password。
  • -pwdf string：密码文件。
  • -rf string：写入 sshkey 文件的 redis 文件，例如：-rf id_rsa.pub。
  • -rs string：写入 cron 文件的 redis shell，例如：-rs 192.168.1.1:6666。
  • -sc string：ms17 shellcode，例如：-sc add。
  • -silent：静默扫描。
  • -socks5 string：设置 socks5 代理，将用于 tcp 连接，超时设置将无效。
  • -sshkey string：sshkey 文件，例如：id_rsa。
  • -t int：线程数（默认值：600）。
  • -time int：设置超时时间（默认值：3；单位：秒）。
  • -top int：显示存活数量排名（默认值：10）。
  • -u string：url。
  • -uf string：url 文件。
  • -user string：用户名。
  • -usera string：添加基于默认用户的用户，例如：-usera user。
  • -userf string：用户名文件。
  • -wmi：启动 wmi。
  • -wt int：设置 web 超时时间（默认值：5；单位：秒）。
