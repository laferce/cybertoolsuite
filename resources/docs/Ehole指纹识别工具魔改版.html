Ehole指纹识别工具魔改版 帮助文档

简介

EHole 是一款用于识别资产中重点系统指纹的工具。在红队作战中，信息收集是必不可少的环节。EHole 旨在帮助红队人员在信息收集期间，能够快速从 C 段、大量杂乱的资产中精准定位到易被攻击的系统（如 OA、VPN、WebLogic 等），从而实施进一步攻击。

用法

EHole_windows_amd64.exe [命令] [选项]
命令
  • EHole_windows_amd64.exe finger：EHole 的指纹识别模块。
    • -f, --fip：从 FOFA 提取资产，进行指纹识别，仅支持 IP 段。
    • -s, --fofa：从 FOFA 提取资产，进行指纹识别，支持 FOFA 所有语法。
    • -a, --hip：从 Hunter 提取资产，进行指纹识别，仅支持 IP 段。
    • -b, --hunter：从 Hunter 提取资产，进行指纹识别，支持 Hunter 所有语法。
    • -l, --local：从本地文件读取资产，进行指纹识别，支持带协议或无协议地址。
    • -o, --output：输出结果文件，支持 .json 和 .xlsx 格式。
    • -p, --proxy：设置代理，支持 HTTP/SOCKS5 协议。
    • -t, --thread：指纹识别并发线程数（默认值：100）。
    • -u, --url：对单个 URL 目标进行指纹识别。
  • EHole_windows_amd64.exe fofaext：EHole 的 FOFA 提取模块。
    • -s, --fofa string：从 FOFA 提取资产，进行指纹识别，支持 FOFA 所有语法，默认保存所有结果。
    • -l, --ipfile string：从本地文件读取资产，在 FOFA 搜索，支持大量资产批量处理，默认保存所有结果。
    • -o, --output string：输出结果文件，支持 .xlsx 格式（默认值：results.xlsx）。
  • help：获取任何命令的帮助信息。
选项
  • --config string：指定配置文件路径（默认值：$HOME/.ehole.yaml）。
  • -h, --help：显示 ehole 的帮助信息。
  • -t, --toggle：切换帮助信息。
