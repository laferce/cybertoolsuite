APP信息收集探测工具 帮助文档

python app.py [命令] [选项]
命令
  • python app.py android：Android 应用程序安全审计：提取 APK 签名信息（META-INF/）；分析 AndroidManifest.xml 权限声明。
    • -i, --inputs TEXT：输入待扫描的 APK/DEX 文件或下载地址[必填]。
    • -r, --rules TEXT：输入临时扫描规则。
    • -s, --sniffer：启用网络流量嗅探（默认开启）。
    • -n, --no-resource：忽略资源文件（包括网络嗅探）。
    • -a, --all：输出所有符合规则的内容（默认开启）。
    • -t, --threads INT：设置并发线程数（默认值：10）。
    • -o, --output TEXT：指定结果输出目录。
    • -p, --package TEXT：指定待扫描的包名。
    • --help：显示帮助信息并退出程序。
  • python app.py ios：iOS 应用程序安全审计：通过备份文件解析敏感数据；提取 iOS 应用脱壳二进制文件。
    • -i, --inputs TEXT：输入待扫描的 IPA/ELF 文件或下载地址（暂不支持 App Store）[必填]。
    • -r, --rules TEXT：输入临时扫描规则。
    • -s, --sniffer：启用网络流量嗅探（默认开启）。
    • -n, --no-resource：忽略资源文件（包括网络嗅探）。
    • -a, --all：输出所有符合规则的内容（默认开启）。
    • -t, --threads INT：设置并发线程数（默认值：10）。
    • -o, --output TEXT：指定结果输出目录。
    • --help：显示帮助信息并退出程序。
  • python app.py web：Web 应用程序安全审计：追踪用户会话凭证；检测 XSS 漏洞存储型载荷。
    • -i, --inputs TEXT：输入待扫描的网站目录、文件或下载地址[必填]。
    • -r, --rules TEXT：输入临时扫描规则。
    • -s, --sniffer：启用网络流量嗅探（默认开启）。
    • -n, --no-resource：忽略资源文件（包括网络嗅探）。
    • -a, --all：输出所有符合规则的内容（默认开启）。
    • -t, --threads INT：设置并发线程数（默认值：10）。
    • -o, --output TEXT：目录。
    • --help：显示帮助信息并退出程序。
选项
  • --help：显示帮助信息并退出程序。
