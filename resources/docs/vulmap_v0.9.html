vulmap 帮助文档

用法

python3 vulmap.py [选项]
目标选项
  • -u URL, --url URL：指定目标 URL 地址（例如：-u "http://example.com"）。
  • -f FILE, --file FILE：指定包含多个目标 URL 地址的文件（例如：-f "list.txt"）。
  • --fofa keyword：调用 FOFA API 扫描（例如：--fofa "app=Apache-Shiro"）。
  • --shodan keyword：调用 Shodan API 扫描（例如：--shodan "Shiro"）。
模式选项
  • -a APP [APP ...]：指定扫描的应用类型（例如：-a "tomcat"；支持多个应用）。
通用选项
  • -t NUM, --thread NUM：扫描线程数（默认值：10）。
  • --dnslog server：DNS 日志服务器（可选：hyuga、dnslog、ceye；默认自动选择）。
  • --output-text file：结果导出为TXT文件（例如："result.txt"）。
  • --output-json file：结果导出为JSON文件（例如："result.json"）。
  • --proxy-socks SOCKS：Socks 代理（例如：--proxy-socks 127.0.0.1:1080）。
  • --proxy-http HTTP：HTTP代理（例如：--proxy-http 127.0.0.1:8080）。
  • --fofa-size SIZE：FOFA 查询目标数量（默认值：100，范围从 1 到 10000）。
  • --user-agent UA：自定义 User-Agent 请求头。
  • --delay DELAY：检测延迟秒数（默认值：0）。
  • --timeout TIMEOUT：扫描超时秒数（默认值：10）。
  • --list：显示支持的漏洞列表。
  • --debug：调试模式：显示 EXP 请求/响应，POC 输出漏洞列表。
  • --check：存活检测开关（默认值：on；可选：on、off）。
支持扫描的应用类型
  • all、activemq、flink、shiro、solr、struts2、tomcat、unomi、drupal、elasticsearch、fastjson、jenkins、laravel、nexus、weblogic、jboss、spring、thinkphp、druid、exchange、nodejs、saltstack、vmware、bigip、ofbiz、coremail、ecology、eyou、qianxin、ruijie
示例
  • python3 vulmap.py -u http://example.com
  • python3 vulmap.py -u http://example.com -a struts2
  • python3 vulmap.py -f list.txt -a weblogic -t 20
  • python3 vulmap.py -f list.txt --output-json results.json
  • python3 vulmap.py --fofa "app=Apache-Shiro"
