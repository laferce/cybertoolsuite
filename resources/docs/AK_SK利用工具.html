AK_SK利用工具 帮助文档

  1. 基本用法：
    java -jar aksktool.jar [ak sk]
    说明：使用访问密钥（ak）和签名密钥（sk）进行操作。
  2. 带令牌的用法：
    java -jar aksktool.jar [ak sk token]
    说明：使用访问密钥（ak）、签名密钥（sk）和令牌（token）进行操作。
  3. 指定区域和产品：
    java -jar aksktool.jar [ak sk region=xxx product=oss,cos]
    说明：指定区域（region）和产品（如 oss、cos）进行操作。
  4. 七牛云用法：
    java -jar aksktool.jar [ak sk bucket_name bucket_url]
    说明：针对七牛云（qiniu），使用访问密钥（ak）、签名密钥（sk）、存储桶名称（bucket_name）和存储桶 URL（bucket_url）进行操作。
