FastJson检测利用工具 帮助文档

功能

批量检测是否存在 Fastjson/Jackson 漏洞（未使用 -f 时，默认检测 Fastjson）。

用法

JsonExp.exe -u [目标] -l [LDAP地址]
JsonExp.exe -uf [url.txt] -l [LDAP地址]
JsonExp.exe -req [request.txt] -l [LDAP地址]
可选选项
  • -h, --help：显示帮助信息并退出。
  • -uf URLFILE, --urlfile URLFILE：[可选] 指定批量目标（文本路径）。
  • -u URL, --url URL：[可选] 指定目标。
  • -req REQUEST, --request REQUEST：[可选] 指定请求包，默认为 HTTP 协议发送请求。
  • -to TIMEOUT, --timeout TIMEOUT：[非必须] 指定请求超时时长（默认值：5；单位：秒）。
  • -f FILE, --file FILE：[非必须] 指定 payload（文本路径；默认值：template/fastjson.txt）。
  • -t TYPE, --type TYPE：[非必须] 指定请求类型为 get 或 post（默认值：post）。
  • -c COOKIE, --cookie COOKIE：[非必须] 设置 Cookie 值。
  • -l LDAP, --ldap LDAP：[可选] 指定 LDAP 地址，ip:端口/路径 或 域名:端口/路径。
  • -r RMI, --rmi RMI：[可选] 指定 RMI 地址，ip:端口 或 域名:端口。
  • -pro PROTOCOL, --protocol PROTOCOL：[可选] 指定请求包的协议类型，需结合 -req 选项使用（默认值：http）。
  • -proxy PROXY, --proxy PROXY：[可选] 设置代理，例：--proxy http://127.0.0.1:8080。
