ENScan 帮助文档

用法

enscan-0.0.15-windows-amd64.exe [选项]
选项
  • -api：是否启用 api 模式。
  • -branch：查询分支机构（分公司）信息。
  • -client string：客户端模式通道，例如：task。
  • -debug：是否显示 debug 详细信息。
  • -deep int：递归搜索 n 层公司（默认值：1）。
  • -delay int：设置最大延迟时间（秒），实际延迟将在 1 到 n 之间随机选择。
  • -email：获取电子邮件信息。
  • -f string：批量查询，文本按行分隔。
  • -field string：获取指定字段信息，例如：icp。
  • -hold：是否查询控股公司。
  • -i string：公司 PID。
  • -invest float：投资比例，例如：100。
  • -is-branch：深度查询分支机构信息（数据量可能很大）。
  • -is-group：查询关键词是否为集团。
  • -is-merge：合并导出。
  • -is-pid：批量查询文件是否为公司 PID。
  • -is-show：是否展示信息输出（默认值：true）。
  • -json：以 json 格式导出。
  • -n string：关键词，例如：小米。
  • -no-merge：批量查询时【取消】合并导出。
  • -o string：结果输出的文件夹位置（可选）。
  • -proxy string：设置代理。
  • -supplier：是否查询供应商信息。
  • -timeout int：每个请求的默认超时时间（默认值：1；单位：分）。
  • -type string：API 类型，例如：qcc（默认值：aqc）。
  • -uncertain-invest：包括未公示投资公司（无法确定占股比例）。
  • -v：显示版本信息。
