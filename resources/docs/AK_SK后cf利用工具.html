AK_SK后cf利用工具 帮助文档

用法

cf.exe [命令]|[选项]
命令
  • cf.exe about：关于作者 (About me)
  • cf.exe config：配置云服务商的访问密钥
    • cf.exe config del：删除访问密钥。
    • cf.exe config ls：列出已配置过的访问密钥。
      • 选项
        • -a, --all：查询全部数据。
    • cf.exe config mf：修改已配置过的访问密钥。
    • cf.exe config query：查询访问凭证所属厂商。
      • 选项
        • -a, --AccessKeyId string：输入要查询的访问凭证 ID。
        • -s, --SecretAccessKey string：输入要查询的访问凭证密钥。
        • -t, --SessionToken string：输入要查询的访问凭证会话令牌。
    • cf.exe config scan：扫描本地访问密钥。
      • 选项
        • -a, --all：查询全部数据。
    • cf.exe config sw：切换访问密钥。
  • cf.exe alibaba：执行与阿里云相关的操作
    • cf.exe alibaba console：一键接管控制台。
      • 命令
        • cf.exe alibaba console cancel：取消接管控制台。
        • cf.exe alibaba console ls：查看接管控制台的信息。
      • 选项
        • -a, --accessKey：创建 AccessKey。
        • -u, --userName string：指定用户名（默认值：crossfire）。
    • cf.exe alibaba ecs：执行弹性计算服务相关操作。
      • 命令
        • cf.exe alibaba ecs exec：在实例上执行命令。
        • cf.exe alibaba ecs imageShare：共享实例镜像。
        • cf.exe alibaba ecs ls：列出所有的实例。
      • 选项
        • --flushCache：刷新缓存，不使用缓存数据。
    • cf.exe alibaba ls：列出当前凭证下的 OSS/ECS/RDS 资源。
      • 选项
        • -a, --allRegions：扫描所有区域，包含私有区域。
        • --flushCache：刷新缓存，不使用缓存数据。
        • -r, --region string：指定区域 ID（默认值：all）。
    • cf.exe alibaba oss：执行对象存储相关操作。
      • 命令
        • cf.exe alibaba oss ls：列出所有的存储桶。
        • cf.exe alibaba oss obj：执行与对象相关的操作。
    • cf.exe alibaba perm：列出当前凭证的权限。
    • cf.exe alibaba rds：执行云数据库相关操作。
      • 命令
        • cf.exe alibaba rds account：添加云数据库账号。
        • cf.exe alibaba rds ls：列出所有的云数据库。
        • cf.exe alibaba rds public：将云数据库设置为公开访问。
        • cf.exe alibaba rds whiteList：为 RDS 添加白名单。
      • 选项
        • --flushCache：刷新缓存，不使用缓存数据。
    • cf.exe alibaba regions：列出可用区域。
      • 命令
        • cf.exe alibaba regions ecs：列出阿里云 ECS 的区域。
        • cf.exe alibaba regions rds：列出阿里云 RDS 的区域。
  • cf.exe aws：执行与亚马逊云相关的操作
    • cf.exe aws ec2：执行与弹性计算服务相关的操作。
      • 命令
        • cf.exe aws ec2 ls：列出所有的实例。
      • 选项
        • --flushCache：刷新缓存，不使用缓存数据。
    • cf.exe aws regions：列出可用区域。
      • 命令
        • cf.exe aws regions ec2：列出 aws ec2 的区域。
    • cf.exe aws s3：执行与对象存储相关的操作。
      • 命令
        • cf.exe aws s3 ls：列出所有的存储桶。
  • cf.exe huawei：执行与华为云相关的操作
    • cf.exe huawei console：一键接管控制台。
      • 命令
        • cf.exe huawei console cancel：取消接管控制台。
        • cf.exe huawei console ls：查看接管控制台的信息。
      • 选项
        • -u, --userName string：指定用户名（默认值：crossfire）。
    • cf.exe huawei obs：执行与对象存储相关的操作。
      • 命令
        • cf.exe huawei obs ls：列出所有的存储桶。
    • cf.exe huawei perm：列出当前凭证的权限。
  • cf.exe tencent：执行与腾讯云相关的操作
    • cf.exe tencent console：一键接管控制台。
      • 命令
        • cf.exe tencent console cancel：取消接管控制台。
        • cf.exe tencent console ls：查看接管控制台的信息。
      • 选项
        • -u, --userName string：指定用户名（默认值：crossfire）。
    • cf.exe tencent cvm：执行与弹性计算服务相关的操作。
      • 命令
        • cf.exe tencent cvm exec：在实例上执行命令。
        • cf.exe tencent cvm ls：列出所有的实例。
      • 选项
        • --flushCache：刷新缓存，不使用缓存数据。
    • cf.exe tencent lh：执行与轻量计算服务相关的操作。
      • 命令
        • cf.exe tencent lh exec：在实例上执行命令。
        • cf.exe tencent lh ls：列出所有的实例。
      • 选项
        • -i, --InstanceId string：指定实例 ID。
        • --flushCache：刷新缓存，不使用缓存数据。
    • cf.exe tencent perm：列出当前凭证的权限。
    • cf.exe tencent regions：列出可用区域。
      • 命令
        • cf.exe tencent regions cvm：列出腾讯云 CVM 的区域。
    • cf.exe tencent uninstall：一键卸载云镜。
      • 选项
        • -u, --UUID string：指定云镜 UUID。
  • cf.exe upgrade：更新 cf 到最新版本
  • cf.exe version：输出 cf 的版本和更新时间
选项
  • -h, --help：显示 cf 的帮助信息。
  • --logLevel string：设置日志等级，可选值为[trace|debug|info|warn|error|fatal|panic]（默认值：info）。
