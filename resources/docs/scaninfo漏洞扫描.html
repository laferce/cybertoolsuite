scaninfo漏洞扫描 帮助文档

说明

本工具只做探测没有提供利用方式只供学习，请遵守国家网络安全。
扫描的时候可以手动停止结果会自动保存，-h 查看使用说明，-show 查看支持的模块。
漏洞和指纹扫描的时候可能最后几个任务很慢，是因为弱口令爆破，可以手动 control+c 结束.
WARNING：默认使用TOP100端口，可以 -p 指定端口或者使用选项 -t1000 扫描 TOP1000 端口。

用法

scaninfo_windows_x64.exe [选项]
选项列表
  • -ei value：设置要排除的 IP 范围。
  • -ep value：设置要排除的端口范围。
  • -ff string：为指纹扫描自定义指定文件。
  • -i value：设置域名和 IP 地址。
  • -l string：输入包含 IP 的文件。
  • -m string：选择扫描类型，例如：-m ssh（默认值：all）。
  • -n int：端口扫描的线程数，范围为 1 到 2000（默认值：900）。
  • -np：对主机进行 ping 扫描以检测是否在线。
  • -o string：保存扫描结果的文件（默认值：result）。
  • -p value：设置要扫描的端口范围，默认为前 100 个常见端口。
  • -pt int：ICMP 扫描的线程数（默认值：100）。
  • -show：显示扫描类型列表。
  • -t float：扫描端口时的 TCP 连接超时时间（默认值：0.5；单位：秒）。
  • -t1000：扫描前 1000 个常见端口。
  • -uf string：输入用于 Web 标题扫描的 URL 文件。
  • -vt int：Web 和漏洞扫描的线程数（默认值：500）。
